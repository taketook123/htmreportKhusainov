Селектори в CSS

Селектори - це цеглинки CSS, які використовуються для вибору елементів HTML, до яких будуть застосовані стилі.

Існує 5 основних типів селекторів:

  • **Селектори типу:** Вибирають елементи HTML за їх тегом.
  • **Селектори класу:** Вибирають елементи HTML за їх класом.
  • **Селектори ID:** Вибирають елементи HTML за їх ID.
  • **Нащадкові селектори:** Вибирають елементи HTML, які є нащадками інших елементів.
  • **Селектори сусідніх елементів:** Вибирають елементи HTML, які знаходяться поруч з іншими елементами.

Також існують комбіновані селектори, які дозволяють поєднувати різні типи селекторів для більш точного вибору елементів.
