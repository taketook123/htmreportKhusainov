Опис предметного середовища

CSS (Cascading Style Sheets)

  • Мова стилів, що використовується для візуального оформлення веб-сторінок.
  • Дозволяє задавати шрифти, кольори, розміри, розташування елементів та багато іншого.
  • Робить веб-сторінки більш привабливими та зручними для користувачів.
  • Є невід'ємною частиною сучасного веб-розробки.

Figma

  • Векторний редактор, призначений для створення інтерфейсів та веб-застосунків.
  • Дозволяє працювати над проектами в команді.
  • Має безліч інструментів для створення UI/UX дизайну.
  • Безкоштовна версія доступна для некомерційного використання.
