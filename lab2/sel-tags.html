Селектори тегів в CSS

Селектори тегів - це найпростіший тип селекторів, які використовуються для вибору елементів HTML за їх тегом.

Щоб використовувати селектор тега, просто вкажіть ім'я тегу в CSS-коді.

Приклад:

	<style>
	h1 {
	  color: red;
	  font-size: 20px;
	}
	</style>
	<button color="red" fontsize="20px"> КНОПКА НАТИВНА </button>

Цей код CSS зробить всі заголовки h1 червоними та збільшить їх розмір до 20 пікселів.

Селектори тегів можна використовувати для стилізації будь-яких елементів HTML, наприклад, заголовків, абзаців, посилань, зображень, списків тощо.
