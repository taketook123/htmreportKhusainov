Завдання 1. Підстановка:

Підстановка в JavaScript - це метод вставки текстових значень в інші рядки або шаблони. Використовується для динамічного створення HTML-коду, форматування рядків, об'єднання даних. Приклад коду: 

	    const user = {
	      name: "Дмитро",
	      age: 10
	    };

	    const messageElement = document.getElementById('message');

	    // Шаблонні літерали
	    messageElement.innerHTML = `
	      <h1>Вітаємо, ${user.name}!</h1>
	      <p>Вам ${user.age} років.</p>

	    `;
