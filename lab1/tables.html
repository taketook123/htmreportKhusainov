Таблиці

Таблиці використовуються для організування даних у рядки та стовпці. HTML пропонує набір тегів для створення таблиць з різними функціональними можливостями.

Основні теги для таблиць

Назва тегу Опис
<caption> Визначає заголовок таблиці, який відображається над нею.
<colgroup> Групує стовпці таблиці для спільного форматування.
<thead> Визначає заголовки стовпців таблиці, які відображаються зверху.
<tbody> Містить основні дані таблиці, представлені у рядках та комірках.
<tr> Створює один рядок таблиці.
<td> Створює звичайну комірку таблиці для розміщення даних.
<th> Створює комірку-заголовок таблиці.
<tfoot> Використовується для розміщення підсумкових рядків внизу таблиці.
