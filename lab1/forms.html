Форми

Веб-форма (форма) — це елемент веб-сторінки, який дає користувачам можливість вводити інформацію та надсилати її на сервер для подальшої обробки. За створення форми в HTML-документі відповідає тег <form>, в якому містяться всі інші необхідні теги.

Основні теги для форм

  • **<form>** - Визначає контейнер для елементів форми.
  • **<input>** - Використовується для створення різних елементів вводу, таких як текстові поля, поля пароля, кнопки вибору і т.д.
  • **<select>** - Використовується для створення випадаючих списків.
  • **<fieldset>** - Використовується для групування елементів форми.
  • **<textarea>** - Використовується для створення багаторядкових текстових полів.
  • **<button>** - Використовується для створення кнопок.
  • **<label>** - Використовується для зв'язування текстового описання з елементом форми.
